Февраль 2012 — Сентябрь 2013

Лаборатория спутниковой океанографии (SOLab)

Должность: Программист. Подробнее

Проекты:

  1. SATIN (SATellite Data Search and Manage INformation Portal) — веб-сайт, предназначенный для доступа к данным дистанционного зондирования океана.

    Используемые технологии: js, jQuery, Bootstrap, CSS, ASP .NET, LFTP, Mercurial.

  2. Сайт SOLab version 2.0 — Сайт Лаборатории спутниковой океанографии.

    Используемые технологии: Django, Django-cms, jQuery, 960px, CSS, HTML mMcrodata, Mercurial.

  3. Polar Lows — проект, посвященный созданию климатологии полярных циклонов в северных моряx.

    Используемые технологии: Matlab, Python, Django, Geoserver, Django-tastypie, PostGIS, Sass, Zurb Foundation Framework, CoffeeScript, Mercurial, GoogleMaps API, jQuery.

    В результате проекта получено два патента:

    • Программа ЭВМ "Метод построения визуализации траектории движения полярных циклонов в высоких широтах по данным спутниковых пассивных микроволновых радиометров (VIrTuAL)";
    • Программа ЭВМ "Полуавтоматический метод расчета параметров полярных циклонов в высоких широтах (SIMPLe)".

  4. Arctic Monitor — прототип cистемы оперативного мониторинга Арктики на основе спутниковых данных.

    Используемые технологии: Django, Python, GeoDjango, GDAL, GeoServer, GeoWebCache, PostGIS, Sass, Zurb Foundation Framework, CoffeeScript и OpenLayers.

  5. SIOWS (Storm Ice Oil Watch System) — Система мониторинга штормов, ледового покрова и нефтянных разливов.

    Проект разрабатывается совместно с французской компанией diateam.Занимаюсь разработкой front-end'a.

Приобретенные в ходе работы навыки:
  • Работа в команде с использованием распределенных систем контроля версий (git и mercurial) и системы отслеживания ошибок (YouTrack);
  • Front-end: HTML5, CSS3, Sass, Less, CoffeeScript, Zurb Foundation, Bootstrap, jQuery UI, jQuery Mobile;
  • Back-end: Django (+ различные модули), PostGIS, MSSQL;
  • Другое: LFTP (автоматизация закачки спутниковых данных);
  • Настройка окружения для Django на основе nginx, gunicorn и supervisor.
